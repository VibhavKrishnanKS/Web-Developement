**index.html** - This must always be the main page of any website  

```HTML
<!DOCTYPE html> - This Document uses HTML
<html></html> - Parent element for both the head and body element
<head></head> - This is basically for the things that are not visible in the browser window
  1. This contains the page title
  2. Description about the page
  3. Link to the CSS Stylesheets
  4. Not rendered in the main part of the browser
<body></body> - All the elements that we see inside the Web Browser will be inside this body element 
  1. Content we want to be rendered goes inside the body element 
```
